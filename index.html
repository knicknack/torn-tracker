About

A small private dashboard for foreign-shop merching. Stock data is crowd-sourced from YATA. Traders, shit list, inventory, and logbook entries are stored in a shared JSONBin so two people can share the same view. The Torn API is used for usernames, market prices, online status, and location stamps. Nothing is logged to any server beyond the shared list contents on JSONBin, which only people with the Master Key can read.
